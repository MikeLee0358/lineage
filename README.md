# 天堂衝裝模擬器

## Live Demo
- [v2](https://lineage-v2.netlify.app/):
  實務歷練後，專注於`衝裝備功能`、`架構之美`。

- [v1](https://mikelee0358.github.io/LineageEmulator/#/login):
  新手時期作品，`什麼都想要` (多角色、音效、未來可能性、遊戲擴充性...)。

## 目的 
  個人的 Web 技術練功場，記錄階段性成長。

## 功能
  - 點擊卷軸 -> 要強化的防具 / 武器 -> 成功 or 失敗。
  - 角色講話時觸發不同事件。（強化機率上升、下降、無效化、冷笑話..）。
  
  - v2 (軟體業工作之後):
    - 導入 ESCSS，優化整體架構。
    - 單一角色，專注衝裝備功能。
    - 使用 playwright 做 E2E 測試。

  - v1 (軟體業工作之前):
    - 多種角色，對應不同主題音樂。
    - 使用 Cypress 做 E2E 測試。

## 備註
  v1 只剩下靜態頁面供回憶，Git 紀錄不再此專案，且被早期的我刪除了..。

## 使用技術
  - ESCSS。
  - Cypress、Playwright。
  - CSS、SCSS、Tailwind。
  - Vue 3、Vue router、Pinia。
